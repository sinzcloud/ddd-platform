## 使用示例

```java
@RestController
@RequestMapping("/api/test")
public class TestController {
    
    // 基于IP限流：每分钟最多10次
    @GetMapping("/test1")
    @RateLimiter(limit = 10, duration = 60)
    public Result<String> test1() {
        return Result.success("test1");
    }
    
    // 基于用户ID限流：每分钟最多5次
    @GetMapping("/test2")
    @RateLimiter(key = "'user:' + #userId", limit = 5, duration = 60)
    public Result<String> test2(@RequestParam Long userId) {
        return Result.success("test2");
    }
    
    // 基于方法名限流：每秒最多2次
    @GetMapping("/test3")
    @RateLimiter(key = "'method:' + #method", limit = 2, duration = 1, timeUnit = TimeUnit.SECONDS)
    public Result<String> test3() {
        return Result.success("test3");
    }
}
```
