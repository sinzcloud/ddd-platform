### 八、常见问题排查

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| `Could not find artifact` | 模块未安装 | 执行 `mvn clean install -DskipTests` |
| 测试失败 | 断言错误 | 查看 surefire-reports 中的错误信息 |
| 启动失败 | 端口占用 | 修改 application.yml 中的端口 |
| 接口 403 | Token无效 | 重新登录获取新 Token |
| 接口 404 | 路径错误 | 检查 URL 是否正确 |
| 中文乱码 | 编码问题 | 设置 console charset 为 GBK |
| MySQL 连接失败 | MySQL未启动 | `net start MySQL` |
| Redis 连接失败 | Redis未启动 | 启动 redis-server |

### 九、测试报告查看

#### 9.1 报告位置

```
ddd-domain/target/surefire-reports/
ddd-application/target/surefire-reports/
ddd-infrastructure/target/surefire-reports/
ddd-interfaces/target/surefire-reports/
ddd-bootstrap/target/surefire-reports/
```

#### 9.2 快速查看

```bash
# 查看所有测试结果汇总
findstr "Tests run:" */target/surefire-reports/*.txt

# 查看失败测试
findstr "FAILURE\|ERROR" */target/surefire-reports/*.txt
```

#### 9.3 报告解读

```
Tests run: 12, Failures: 0, Errors: 0, Skipped: 0
```

- **Tests run**: 运行的测试数量
- **Failures**: 断言失败（业务逻辑错误）→ 必须为 0
- **Errors**: 代码异常（程序错误）→ 必须为 0
- **Skipped**: 跳过的测试 → 通常为 0
