## 分布式锁的实际使用场景

### 1. 在Controller中使用

**ddd-interfaces/src/main/java/com/ddd/platform/interfaces/controller/UserController.java**

```java
@RestController
@RequestMapping("/api/users")
@RequiredArgsConstructor
@Tag(name = "用户管理", description = "用户相关接口")
public class UserController {
    
    private final UserApplicationService userApplicationService;
    private final DistributedLock distributedLock;
    
    // 方式1：使用注解方式
    @PutMapping("/{userId}/update")
    @DistributedLockAnnotation(key = "'user:update:' + #userId", waitTime = 3, leaseTime = 10)
    @Operation(summary = "更新用户信息", description = "防止并发更新")
    public Result<Void> updateUser(@PathVariable Long userId, @RequestBody UserDTO userDTO) {
        userApplicationService.updateUser(userId, userDTO);
        return Result.success();
    }
    
    // 方式2：编程式使用
    @PostMapping("/deduct-points")
    @Operation(summary = "扣除用户积分", description = "使用分布式锁防止积分超扣")
    public Result<Void> deductPoints(@RequestParam Long userId, @RequestParam Integer points) {
        String lockKey = "user:points:" + userId;
        
        // 获取锁，等待3秒，持有10秒
        boolean locked = distributedLock.tryLock(lockKey, 3, 10, TimeUnit.SECONDS);
        if (!locked) {
            throw new BizException("操作太频繁，请稍后再试");
        }
        
        try {
            // 执行积分扣除业务
            userApplicationService.deductPoints(userId, points);
            return Result.success();
        } finally {
            distributedLock.unlock(lockKey);
        }
    }
}
```

### 2. 在Service中使用

**ddd-application/src/main/java/com/ddd/platform/application/service/UserApplicationService.java**

```java
@Service
@RequiredArgsConstructor
public class UserApplicationService {
    
    private final UserRepository userRepository;
    private final DistributedLock distributedLock;
    
    // 场景1：用户注册 - 防止同一个用户名并发注册
    public UserDTO registerUser(RegisterCommand command) {
        String lockKey = "user:register:" + command.getUsername();
        
        return distributedLock.executeWithLock(lockKey, 3, 10, TimeUnit.SECONDS, () -> {
            // 检查用户是否存在
            if (userRepository.existsByUsername(command.getUsername())) {
                throw new BizException(ErrorCode.USERNAME_EXIST);
            }
            
            // 创建用户
            User user = new User();
            user.setUsername(command.getUsername());
            user.setPassword(passwordEncoder.encode(command.getPassword()));
            user.setEmail(command.getEmail());
            user.setStatus(1);
            
            user = userRepository.save(user);
            return convertToDTO(user);
        });
    }
    
    // 场景2：积分扣除 - 防止并发超扣
    public void deductPoints(Long userId, Integer points) {
        String lockKey = "user:points:" + userId;
        
        distributedLock.executeWithLock(lockKey, 3, 5, TimeUnit.SECONDS, () -> {
            User user = userRepository.findById(userId)
                .orElseThrow(() -> new BizException(ErrorCode.USER_NOT_FOUND));
            
            // 检查积分是否足够
            if (user.getPoints() < points) {
                throw new BizException("积分不足");
            }
            
            // 扣除积分
            user.setPoints(user.getPoints() - points);
            userRepository.save(user);
            
            return null;
        });
    }
    
    // 场景3：库存扣减（秒杀场景）
    public void deductStock(Long productId, Integer quantity) {
        String lockKey = "product:stock:" + productId;
        
        distributedLock.executeWithLock(lockKey, 5, 10, TimeUnit.SECONDS, () -> {
            // 查询库存
            Product product = productRepository.findById(productId);
            if (product.getStock() < quantity) {
                throw new BizException("库存不足");
            }
            
            // 扣减库存
            product.setStock(product.getStock() - quantity);
            productRepository.save(product);
            
            // 创建订单
            createOrder(productId, quantity);
            
            return null;
        });
    }
}
```

### 3. 订单创建场景

**ddd-application/src/main/java/com/ddd/platform/application/service/OrderApplicationService.java**

```java
@Service
@RequiredArgsConstructor
public class OrderApplicationService {
    
    private final DistributedLock distributedLock;
    private final OrderRepository orderRepository;
    private final ProductRepository productRepository;
    
    // 场景：创建订单 - 防止重复下单
    public void createOrder(CreateOrderCommand command) {
        // 使用用户ID + 商品ID作为锁key，防止同一个用户对同一个商品重复下单
        String lockKey = "order:create:" + command.getUserId() + ":" + command.getProductId();
        
        distributedLock.executeWithLock(lockKey, 5, 30, TimeUnit.SECONDS, () -> {
            // 检查是否已经有未支付的订单
            boolean hasPendingOrder = orderRepository.existsPendingOrder(
                command.getUserId(), command.getProductId());
            if (hasPendingOrder) {
                throw new BizException("您已有未支付的订单，请勿重复下单");
            }
            
            // 检查库存
            Product product = productRepository.findById(command.getProductId());
            if (product.getStock() < command.getQuantity()) {
                throw new BizException("库存不足");
            }
            
            // 扣减库存
            product.setStock(product.getStock() - command.getQuantity());
            productRepository.save(product);
            
            // 创建订单
            Order order = new Order();
            order.setOrderNo(generateOrderNo());
            order.setUserId(command.getUserId());
            order.setProductId(command.getProductId());
            order.setQuantity(command.getQuantity());
            order.setAmount(product.getPrice() * command.getQuantity());
            order.setStatus(0); // 待支付
            orderRepository.save(order);
            
            return null;
        });
    }
    
    private String generateOrderNo() {
        return "ORD" + System.currentTimeMillis() + UUID.randomUUID().toString().substring(0, 8);
    }
}
```

    @Scheduled(cron = "0 0/30 * * * ?") // 每30分钟执行一次
    public void syncData() {
        String lockKey = "job:sync:data";
        
        // 尝试获取锁，持有时间5分钟
        boolean locked = distributedLock.tryLock(lockKey, 0, 300, TimeUnit.SECONDS);
        if (!locked) {
            log.info("其他实例正在执行同步任务，跳过");
            return;
        }

            // 执行同步逻辑
            doSyncData();
            log.info("数据同步任务完成");
        } finally {
            distributedLock.unlock(lockKey);
        }
    }

    }
}

### 5. 消息消费场景

**ddd-application/src/main/java/com/ddd/platform/application/consumer/OrderConsumer.java**

```java
@Component
@RequiredArgsConstructor
public class OrderConsumer {
    
    private final DistributedLock distributedLock;
    private final OrderService orderService;
    
    // 场景：消息队列消费 - 防止重复消费
    @RabbitListener(queues = "order.queue")
    public void handleOrderMessage(OrderMessage message) {
        String lockKey = "mq:order:" + message.getOrderId();
        
        distributedLock.executeWithLock(lockKey, 3, 10, TimeUnit.SECONDS, () -> {
            // 检查消息是否已处理
            if (orderService.isMessageProcessed(message.getMessageId())) {
                log.info("消息已处理过，跳过: {}", message.getMessageId());
                return null;
            }
            
            // 处理订单
            orderService.processOrder(message.getOrderId());
            
            // 标记消息已处理
            orderService.markMessageProcessed(message.getMessageId());
            
            return null;
        });
    }
}
```

## 常见使用场景总结

| 场景 | 锁Key示例 | 说明 |
|------|-----------|------|
| 用户注册 | `user:register:username` | 防止同一个用户名并发注册 |
| 积分扣除 | `user:points:userId` | 防止积分超扣 |
| 库存扣减 | `product:stock:productId` | 防止库存超卖 |
| 创建订单 | `order:create:userId:productId` | 防止重复下单 |
| 定时任务 | `job:jobName` | 防止多实例重复执行 |
| 消息消费 | `mq:messageId` | 防止消息重复消费 |
| 文件处理 | `file:process:fileId` | 防止同一文件并发处理 |
